Lambdr
======

## Installation
```
npm install lambdr -g
```

## Usage
#### Creating a project

```
lambdr new projectName
```

#### Creating a function
```
lambdr function:create signup
```

#### Deploy all functions
```
lambdr function:deploy
```

#### Create a stage
```
lambdr stage:create staging
```

#### List stages
```
lambdr stage:list
```

#### Remove a stage
```
lambdr stage:remove staging
```
